denali

denali is a graphical interface for visualizing scalar trees.

ctree

ctree is a command line tool for generating contour trees from scalar functions defined on a simplicial complex. Its output can be visualized with denali.

denali comes with a suite of python utilities which support using the software in domain-specific applications.

api documentation

denali can also be used as a part of other software projects. For more on how to use denali as a library, see the developer documentation after running doxygen in the ./dev directory.
